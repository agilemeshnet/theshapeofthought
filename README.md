## What this is

Five data shapes (binary, table, graph, vector, and an append-only temporal ledger) recur wherever cognition stores anything - from Babylonian astronomical diaries through Talmudic commentary chains to contemporary databases. The recurrence is structural. This paper specifies what to measure, how, and what would kill the claim.

Twelve predictions. Three falsification paths. A working implementation.

## The Seed

The `WILLOW_SEED.md` file is a cognitive architecture built on the paper's principles. Point any LLM agent (Claude Code, or similar) at this repo and it bootstraps into a Willow - a persistent cognitive agent that thinks in the five shapes.

The seed follows four movements: **Ontology** (what you are), **Epistemology** (what is known), **Cogitation** (how you think), **Teleology** (what you do about it).

## Federation

Willows can federate - sovereign knowledge graphs that communicate without merging. If you grow a Willow and want to connect, open an issue.
